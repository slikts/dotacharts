#dotacharts

 * Requires [smoke](https://github.com/skadistats/smoke)
 * Creates standalone HTML files
 * Charts:
   * Gold / Minute
   * Experience / Minute
   * Last Hit
 * Very quick and dirty
 * [Example charts](https://slikts.github.io/dotacharts)
 * Usage: `python parse.py [demofile]`
